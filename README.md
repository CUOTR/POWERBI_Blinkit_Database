# EXCEL-POWERBI-Blinkit-Database
# Overview
This file contains a multi-page business intelligence dashboard for Blinkit. It is structured into four main analytical views: Intro, Marketing, Customer & Feedback, and Order & Product.

Source: https://www.kaggle.com/datasets/yashmotiani/blinkit-marketing-and-customer-powerbi-dashbord

# Page 3: Customer & Feedback Analytics
